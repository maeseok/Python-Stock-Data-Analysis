- 현대 포트폴리오 이론
1. 수익률의 표준편차는 자산 가격이 평균값에서 벗어나는 정도, 즉 리스크를 측정하는 방법이다.
EX) 표준편차 5% 수익률 10%면 5~15% 확률이 68.3%, 2표준편차는 95.4, 3표준편차는 99.7%]

2. 효율적 투자선이란 투자자가 인내할 수 있는 리스크 수준에서 최사으이 기대수익률을 제공하는 포트폴리오의 집합
- 효율적 투자선 위에 자산을 투자해야 한다. (포함된 종목 안에서 분산 투자하여 선 위에 있어야 안전)

3. 샤프 지수와 포트폴리오 최적화
- 샤프 지수 : 측정된 위험 단위당 수익률 = (포트폴리오 예상 수익률 - 무위험률) / 수익률의 표준편차
EX) 샤프 지수가 높을수록 위험에 대한 보상이 더 크다.

과거 결과       삼성전자  SK하이닉스  현대자동차 NAVER
샤프 지수 최대 : 48.22      40.17     10.17    0.8
리스크 최소    : 33.8       9.1       35.8     21.1

- 볼린저 밴드 지표
1. 볼린저 밴드 : 주가의 변동이 표준 정규분포를 따른다는 가정에서 주가의 위아래로 밴드를 표시함(기술적분석)

2. 표준 볼린저 밴드 공식 - 95.4% 확률로 주가가 볼린저 밴드 안에 있음
EX) 상단 볼린저 밴드 = 중간 볼린저 밴드 + (2X표준편차)
    중간 볼린저 밴드 = 종가의 20일 이동평균
    하단 볼린저 밴드 = 중간 볼린저 밴드 - (2X표준편차)

3. 볼린저 밴드 지표 : %b
EX) (종가-하단 볼린저 밴드) / (상단 볼린저 밴드-하단 볼린저 밴드)

4. 볼린저 밴드 지표 : 밴드폭 = 상단 볼린저 밴드와 하단 볼린저 밴드 사이의 폭을 의미한다.
EX)(상단 볼린저 밴드 - 하단 볼린저 밴드) / 중간 볼린저 밴드
- 스퀴즈를 확인하는데 유용하다. - 6개월 저점을 기록하는 것을 보고 파악
- 스퀴즈 : 변동성이 극히 낮은 수준까지 떨어져 곧이어 변동성 증가가 발생한 것으로 예상되는 상황
- 강력한 추세 : 스퀴즈로부터 시작 -> 변동성 커지며 밴드폭 수치가 급격히 높아진다.
- 강한 = 하단 볼린저 밴드가 아래로 향하는 모습, 이후 밴드폭 수치 없거나 줄면서 추세 반대쪽 향하면 추세 끝
- 즉 상단 볼린저 밴드와 하단 볼린저 밴드의 사이가 넓어질 때가 강한 추세


- 볼린저 밴드 매매 기법

범주           기술적지표
모멘텀         변화율, 스토캐스틱
추세           선형회귀, MACD
거래량         일중강도, 매집/분산, 현금흐름지표, 거래량가중 MACD
과매수/과매도   CCI, RSI
심리           여론조사선, 풋-콜 비율

-변동성 돌파 매매기법은 주가가 상단 밴드를 상향 돌파할 때 매수하고 주가가 하단 밴드를 하향 이탈할 때 공매도 (우리나라 어려워서 제외)

1. 추세 추종 매매기법
- 상승 추세에 매수하고 하락 추세에 매도하는 기법이다.
- %b + 현금흐름지표(MFI) + 일중강도 로 확증 후 매수/매도하는
EX) 매수 : 주가가 상단 밴드에 접근하여, 지표가 강세를 확증할 때
           (%b>0.8, MFI>80)
    매도 : 주가가 하단 밴드에 접근하여, 지표가 약세를 확증할 때
           (%b<0.2, MFI<20)
1-1 MFI : 중심 가격*거래량 = 현금흐름(가격+거래량을 동시 분석)
EX) 100-(100/(1+긍정적 현금 흐름 / 부정적 현금 흐름))
    긍정적 현금 흐름 : 중심 가격이 전일보다 상승한 날들의 현금 흐름의 합
    부정적 현금 흐름 : 중심 가격이 전일보다 하락한 날들의 현금 흐름의 합

2. 반전 매매 기법
- 주가가 반전하는 지점을 찾아내 매수 또는 매도하는 기법이다.
EX) 매수 : 주가가 하단 밴드 부근에서 W형 패턴을 나타내고, 강세 지표가 확증할 때 매수
           (%b<0.05 ||%>0)
    매도 : 상단 밴드 부근에서 일련의 주가 태그가 일어나며, 약세 지표가 확증할 때 매도
           (%b>0.95 ||%<0)
2-1 일중 강도(||%) 
- 거래 범위에서 종가의 위치를 토대로 주식 종목의 자금 흐름을 설명한다.
- 주가가 하단 볼린저 밴드에 닿을 때 일중 강도율 +면 매수
- 주가가 상단 볼린저 밴드에 닿을 때 일중 강도율 -면 매도
EX) 1 = 종가가 거래 범위 천정권, 0 = 중간, -1 = 바닥권
    일중 강도 = (2*종가-고가-저가)/(고가-저가)*거래량
    일중 강도율 = 일중강도의21일합/거래량의21일합*100
